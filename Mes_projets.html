Accueil

Réalisation d'une application desktop et mobile destinée aux pratiquants d'activitées nautiques (surf, kitesurf, windsurf, planche à voile, voile, ...).

Cette application propose sur son interface toutes les données concernant la météo marine pour plage ou "spot" choisi par son utilisateur.
Les données sont récupérées sur une api distante.
Ces donnéees concernent la météo générale, l'état de la mer (houle, direction de houle, ...) et les données spécifiques au vent (force, direction, rafales).
En plus des données météos transmises à l'utilisateur, ce dernier à la possibilité de se connecter pour participer à un "chat" avec possibilité d'uploader une photo du spot. Ce chat est spécifique au lieu sélectionné.

D'un point de vue technique, cette application est en cours de réalisation. Le fetch et l'affichage des données fonctionnent, à l'heure actuelle, principalement en front-end à l'aide de javascript. Mon objectif est de séparer toute la partie fetch des données et calcul dans un back-end NodeJs et utiliser React pour l'affichage desktop et React Native pour les mobiles.
Pour un dynamisme de l'effet du vent, un layout sera créé à l'aide de canvas et viendra se superposer à une map du spot.
